## 特性

- **一切皆扩展**
积木式搭建自己的“Web3D编辑器”，完全可扩展
- **只开发一次**
一个需求只实现一次，到处复用

- **Web3 Dapp**
拥抱Web3，完全开源

## 优势

- 世界首个搭建Web3D编辑器的低代码平台
- 一切皆可扩展，用户可100%自定义    
目前暂时只开放了“写逻辑代码”的扩展方法，未来将会开放更多的扩展方法
- 拥抱Web3，完全开源
